# Changelog

## [0.2.0] - 2026-04-17

- Supabase opcional con modo degradado local.
- IngestionStore persistente en SQLite con deduplicación SHA-256.
- Endpoint de estado por documento y máquina de estados explícita.
- Parser PDF unificado en PyMuPDF con OCR híbrido por página.
- Panel admin con polling real de documentos.
- Tests unitarios y E2E para parser, store e ingesta.
